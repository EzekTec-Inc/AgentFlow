AgentFlow `Flow` engine routing behavior:
- The **only** key reserved for routing decisions is `"action"`.
- After a node executes, `Flow` checks the `SharedStore` for the `"action"` key.
- If present and a string, it routes to the next node using that value as the edge name (e.g. `action == "replan"` follows the `"replan"` edge).
- If missing or not a string, it defaults to routing via the `"default"` edge.
- Once `Flow` completes, it automatically removes the `"action"` key from the store.
